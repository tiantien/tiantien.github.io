# kvm

- kvm安装 (opens new window)

创建虚拟网桥设备，设备的静态ip设置为现有的IP，网络设备都通过这个网桥转发，不用设置俩个IP

# 增加虚机硬盘

# 1.关闭虚拟机
qemu-img resize working112.qcow2 +200G
fdisk -l
fdisk /dev/vda
reboot
pvcreate /dev/vda3
vgextend centos /dev/vda3
lvextend -L +60G /dev/centos/root
xfs_growfs /dev/centos/root
